# Arxiv Search API

此 API 端點用於搜索 arXiv 上的論文，並返回匹配的結果。

## 端點

`GET api/v1/arvix/search`

### 查詢參數

| 參數名       | 類型    | 必須  | 描述                                                |
| ------------ | ------- | ----- | --------------------------------------------------- |
| `query`      | string  | 否    | 要搜索的關鍵字。默認值為空字符串。                   |
| `max_results`| integer | 否    | 返回的最大結果數量。默認值為 10。                    |

### 返回值

成功響應返回一個包含以下字段的 JSON 對象：

| 字段名       | 類型      | 描述                                               |
| ------------ | --------- | -------------------------------------------------- |
| `success`    | boolean   | 請求是否成功。                                     |
| `data`       | array     | 搜索結果的列表，每個結果是一個對象。                |
| `errors`     | string    | 如果請求失敗，這個字段包含錯誤信息。                 |

`data` 字段中的每個結果對象包含以下信息：

| 字段名       | 類型      | 描述                                               |
| ------------ | --------- | -------------------------------------------------- |
| `title`      | string    | 論文的標題。                                       |
| `authors`    | array     | 論文的作者列表，每個作者是一個字符串。               |
| `summary`    | string    | 論文的摘要。                                       |
| `published`  | string    | 論文的發表日期，格式為 `YYYY-MM-DD`。               |
| `pdf_url`    | string    | 論文的 PDF 下載鏈接。                               |

### 錯誤響應

如果請求失敗，將返回一個包含 `success` 和 `errors` 字段的 JSON 對象，`success` 為 `false`，`errors` 為錯誤信息。
